# LEARNING_ONLINE
Dự án xây dựng một hệ thống Learning Management System (LMS) gồm 3 phần riêng biệt:  Backend — Máy chủ Java Spring Boot. Frontend — Giao diện web dành cho Học viên (Student). Admin — Giao diện web dành cho Giảng viên (Teacher) và Quản trị viên (Admin). Toàn bộ quá trình phát triển được chia làm 7 Phase theo thứ tự từ nền tảng đến hoàn thiện.
